Expand description

Trait that implements conversion of graphs to tensors

This implements a generic method ToTensor::to_tensor for any number type that implements TensorElem, as well as two convenience methods ToTensor::to_tensorf and ToTensor::to_tensorf for FScalar and floating-point Complex numbers, respectively.
