§Basic ZX-calculus rules

These rules always come in triples of functions. For a rule X, there is a function check_X(&g, ...) -> bool which checks whether a rule is applicable at the given vertex or vertices, X_unchecked(&mut g, ...) applies the rule without doing any checking, and X(&mut g, ...) -> bool is the composition of the first two.

Note calling X_unchecked is allowed to make unsound ZX-diagram transformations, or even panic, if check_X doesn’t return true.
